Creating an image
==================

How to create an image with basic NumPy commands : ``np.zeros``, slicing...

This examples show how to create a simple checkerboard.




.. image:: /packages/scikit-image/auto_examples/images/sphx_glr_plot_check_001.png
    :class: sphx-glr-single-img





.. code-block:: python


    import numpy as np
    import matplotlib.pyplot as plt

    check = np.zeros((8, 8))
    check[::2, 1::2] = 1
    check[1::2, ::2] = 1
    plt.matshow(check, cmap='gray')
    plt.show()
